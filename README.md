# Web Automation Underwriting Ritel

Prototype aplikasi underwriting ritel Jasindo berbasis `React + Vite`.

Ruang lingkup utama di repo ini:
- flow produk ritel untuk properti dan kendaraan
- portal nasabah (`Portal Saya`)
- ruang kerja internal (`Ruang Kerja Saya` dan `Tinjauan Internal`)
- konfigurasi partner (`Konfigurasi Partner`)

## Menjalankan lokal

```powershell
npm install
npm run dev
```

## Build produksi

```powershell
npm run build
```

## Catatan

Repo ini dipakai sebagai source code utama untuk deployment Vercel `web-automation-underwriting-ritel`.
